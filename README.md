# Team AI Agent Skills

Shared Claude Code skills for the research team.

## Installation

```
/plugin marketplace add araju-econ/Team-AI-Agent-Skills
```

## Available Skills

| Skill | Description |
|-------|-------------|
| `stata` | Write, run, and debug Stata do-files |
| `qualtrics` | Create and format Qualtrics surveys |
| `read-paper` | Read and summarize economics papers |
| `split-pdf` | Deep-read academic PDFs in structured chunks |
| `compiledeck` | Create and compile Beamer presentations |
| `elegant-slides` | Apply the Elegant DS Beamer theme |
| `review-deck` | Review slide decks for quality |
| `review-academic-slides` | Review academic presentations |
| `explore-folder` | Explore and summarize folder contents |
| `newproject` | Scaffold a new research project |
| `skill-creator` | Create and improve custom skills |

## Updating

Team members get updates automatically when changes are pushed to this repo.
